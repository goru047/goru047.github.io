## 🚀 Quick Start

1. **Extract the ZIP**
   ```bash
   unzip travel-map-blog-part1.zip
   cd travel-map-blog
   ```

2. **Install Jekyll** (if needed)
   ```bash
   gem install bundler jekyll
   ```

3. **Install Dependencies**
   ```bash
   bundle install
   ```

4. **Customize**
   - Edit `_config.yml` with your info
   - Update site title, author, URL
